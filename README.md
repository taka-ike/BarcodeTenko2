# BarcodeTenko2

バーコードリーダーを使用した点呼・スキャン記録管理システム（Windowsネイティブ版）です。

## 特徴
- **高速起動**: .NET 8 WPFによる軽量なネイティブ動作。
- **入力安定性**: バーコードリーダーからの高速な入力を正確にキャッチ。
- **ポータブル運用**: 実行ファイルと同じ階層にデータを保存するため、USBメモリ等での運用が可能。
- **データ互換性**: 既存システムと互換性のあるバイナリ形式（ids_*.bin）とCSV出力をサポート。

## 使い方
1. アプリを起動します。
2. 初回起動時または場所を変更したい場合は、右上の「設定」から場所を選択してください。
3. バーコードをスキャンするか、中央の入力欄に数字を入力してEnterを押します。
4. 履歴は自動的に `data/history.json` および `scans/ids_<場所>.bin` に保存されます。

## データ仕様
- **バイナリ記録**: `scans/ids_<location>.bin` に下5桁の数値を UInt16 Little Endian (2バイト) で追記します。
- **履歴**: `data/history.json` にJSON形式で保存されます。
- **場所設定**: `data/locations.json` を編集することで、選択肢をカスタマイズできます。

## 開発環境
- .NET 8.0
- Visual Studio 2022 / JetBrains Rider
